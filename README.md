# xiami-cli
The command line interface for Xiami

```
xiami login username password
xiami logout
xiami checkin
xiami daily-recommended --download|-d

xiami song 9998 --download|-d
xiami album 9986 --download|-d
xiami collection 9988 --download|-d

xiami my-songs --page=all|-p --download|-d --userid|-uid
xiami my-albums --page=all|-p --download|-d --userid|-uid
xiami my-collections --page=all|-p --download|-d --userid|-uid
```
